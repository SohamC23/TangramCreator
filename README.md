# Tangram Puzzle Generator & 3D-Printed Set

Python project for generating tangram-style puzzles, visualizing solutions, and connecting the digital design process with a physical 3D-printed puzzle set

## About

This project uses computational geometry to generate tangram puzzle layouts and solution visualizations. It was built in Python and uses Matplotlib and Shapely to create and display puzzle arrangements. The project also extends beyond software by including custom tangram piece design in TinkerCAD and physical prototypes produced through 3D printing

## Features

- Generates tangram puzzle layouts using geometric logic
- Visualizes puzzle shapes and matching solutions
- Uses Shapely for shape handling and spatial operations
- Uses Matplotlib for puzzle and solution rendering

## Technologies Used

- Python
- Matplotlib
- Shapely
- TinkerCAD
- 3D Printing

## Project Highlights

- Built a Python-based tangram puzzle generator using computational geometry
- Created puzzle images and solution visualizations for testing and review
- Designed custom tangram pieces in TinkerCAD with dimensions suitable for 3D printing and fed dimensions into the program
- Produced physical puzzle prototypes to connect software output with hands-on design
- Improved puzzle quality by solving generated outputs and refining layout logic
